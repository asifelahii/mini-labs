# Mini Labs

A collection of small frontend experiments and mini projects built to practice JavaScript fundamentals, DOM manipulation, UI design, and clean project organization.

This repository works as a personal frontend lab where I build focused, lightweight projects to strengthen core web development skills through practical implementation.

---

## Live Site

- Landing Page: https://asifelahii.github.io/mini-labs/

---

## Projects

### 1. Random Background Color Generator

A simple interactive web app that changes the background color automatically using JavaScript timer functions.

**Live Demo:** https://asifelahii.github.io/mini-labs/random-bg-color/  
**Source Code:** `projects/random-bg-color/`

#### Key Learning Areas

- DOM manipulation
- JavaScript event handling
- `setInterval()` and `clearInterval()`
- Dynamic style updates
- UI layout and visual polish
- Simple project structuring
